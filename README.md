# winget-pkg-versions

Retrieve latest versions for winget packages.

## Structure

```
manifests/
  a/
    Adobe.Acrobat.Reader.yaml
    Atlassian.Sourcetree.yaml
  b/
    Blender.Blender.yaml
  m/
    Microsoft.Windows.yaml
  ...
```

Content Example:

```
id: Adobe.Acrobat.Reader
version: 22.002.20222
```
